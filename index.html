PWA Practice

PWA(Progressive Web Apps)를 학습하며 진행되는 내용을 기록하는 Repository 입니다.



PWA 구성 요소

  • HTTPS

    • PWA를 제공하기 위해서는 HTTPS가 필수

  • Service Worker

    • 네트워크 사이의 프록시 서버 역할
    • 효과적인 오프라인 서비스 제공
    • 푸시 및 백그라운드 동기화 API 접근 제공
    • 탐색과 리소스 요청을 가로채 수정하거나 리소스를 세부적으로 캐싱
    • Javascript로 Service Worker 생성

  • Web App Manifest

    • 브라우저에서 PWA 여부를 확인하며, 설치 시 어떻게 동작해야하는지 알려주는 JSON 파일
    • 앱 이름, 설명, 아이콘, 색상 등 속성을 가짐
